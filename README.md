# fluffy-octo-winner

## Project Overview
This project is a starting point for a .NET web application. It provides a basic structure for building web apps with ASP.NET Core.

## Features
- ASP.NET Core MVC structure
- Razor Pages for UI
- Bootstrap for styling
- Ready for local development

## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/simonthurman/fluffy-octo-winner.git
   ```
2. Navigate to the project directory:
   ```sh
   cd fluffy-octo-winner
   ```
3. Restore dependencies:
   ```sh
   dotnet restore
   ```
4. Run the application:
   ```sh
   dotnet run
   ```

## Usage
- Access the app at `https://localhost:5001` or as specified in your launch settings.
- Modify pages in the `Pages/` directory to customize the UI.
